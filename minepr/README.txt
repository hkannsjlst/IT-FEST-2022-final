Инструкция для открытия через опен сервер:


1. Скидываем папку minepr в папку domains опен сервера
2. Импортируем базы с файлов sql, два файла с бд - test.sql и mydb.sql
3. По дефолту логин - root, поле пароля пустое. Если у вас другие данные для входа в PHPmyadmin, то придётся почти во всех PHP файлах менять логин и пароль на свои значения 
4. Если вы всё сделали верно, то открываем папку minepr через опен сервер, насладжаемся проектом)
